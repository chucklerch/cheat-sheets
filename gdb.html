GDB

Launching
gdb -q -ex 'break debug' -ex 'run' -w ./app Start gdb with debug breakpoint and run app

Basic Commands
backtrace, bt Display the current call stack (can be used after a runtime error, eg. segfault)
continue, c Continue execution until next breakpoint
step, s Step to next line of code; will step into a function
next, n Execute the next line of code; will step over a function

Display
print (char []) $rax Print contents of register
info address var
Print ([]char) address 		Print contents of variable
