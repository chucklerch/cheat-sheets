Microsoft Terminal Hotkeys

Pane Commands
ALT + SHIFT + - Split Horizontally
ALT + SHIFT + + Split Vertically
ALT + SHIFT + D Split Auto
ALT + Move Focus
ALT + SHIFT + Resize
CTRL + SHIFT + W Close Pane
Font Commands
CTRL + = Increase font size
CTRL + - Decrease font size
CTRL + 0 Reset font size

Misc Commands
CTRL + SHIFT + P Open Command Palette
CTRL + , Open Settings UI
CTRL + ALT + , Open Settings File
CTRL + SHIFT + F Find
